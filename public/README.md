# public (static files)

This folder contains static assets served as-is by the dev server / production build.

Top items

- `favicon.ico`, `favicon.svg` - site icons
- `robots.txt` - robots rules for search engines
- `placeholder.svg` - placeholder image used in the project
- `music/` - background music files (see `music/README.md`)
- `photos/` - photo assets for the gallery (see `photos/README.md`)

Place any files that must be served statically under this folder.
