# src/components/ui

This folder contains small UI primitives and component wrappers used throughout the app.

Examples

- `button.tsx` - styled button component
- `card.tsx` - basic card layout wrapper
- `dialog.tsx`, `sheet.tsx`, `popover.tsx` - overlay components for modals and panels
- `input.tsx`, `textarea.tsx` - form inputs
- `toast.tsx`, `toaster.tsx`, `use-toast.ts` - notification utilities

The remaining files implement common UI patterns (tabs, slider, carousel, table, etc.). Edit these when you need to change a specific UI primitive.
