# src/components

Contains React components used by the app.

Key files/folders

- `BirthdayWebsite.tsx` - top-level birthday landing component (assembles birthday feature pieces).
- `birthday/` - feature-specific components (messages, quiz, animations, music, gallery).
- `ui/` - collection of small UI primitives and shadcn-style components (button, card, modal, etc.).

Check the `birthday/` and `ui/` folders for more detailed README files.
