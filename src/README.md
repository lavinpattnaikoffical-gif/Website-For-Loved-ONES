# src (Source code)

This folder contains the application's source code.

Structure

- `main.tsx` - application entry, mounts React tree.
- `index.css`, `App.css` - global and app-specific styles.
- `App.tsx` - top-level app component.
- `vite-env.d.ts` - Vite TypeScript env types.
- `assets/` - images and other media used by the app.
- `components/` - React components (feature and UI libraries).
- `hooks/` - custom React hooks used across the app.
- `lib/` - small utilities and helpers.
- `pages/` - route/page components (Index, NotFound).

Each subfolder has its own `README.md` describing contained files.
